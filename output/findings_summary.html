Reproduction & Extension: "Smartphones, Online Music Streaming, and Traffic Fatalities"

Reference: Patel, Worsham, Liu & Jena — NBER Working Paper 34866, Feb 2026
Data: NHTSA Fatality Analysis Reporting System (FARS), real daily fatality counts
Analysis date: March 2026  |  Back-of-the-envelope replication check

Executive Summary

We reproduced the paper's core analysis using real FARS data and extended it through 2023. The original 2017–2022 finding largely replicates, but adding 2023 data eliminates the effect entirely. The five 2023 mega-releases show zero association with traffic fatalities.

Model Specification

Outcome: National daily count of fatal motor-vehicle crash fatalities (FARS)

Primary comparison model:
fatalities ~ is_release_day + C(dow) + C(week_of_year) + C(year) + is_holiday
OLS regression at the album–day level. is_release_day = 1 on the album release date, 0 on surrounding days.

Event study model:
fatalities ~ day_m10 + day_m9 + … + day_p0 + … + day_p10 + C(dow) + C(week_of_year) + C(year) + is_holiday
Separate indicator for each day −10 to +10 relative to release. Reference category = day −1.

Fixed effects: Day of week (Mon–Sun), week of year (1–52), calendar year, federal holiday indicator
Event window: ±10 days around each album release
Sample: Top 10 albums by first-day Spotify streams in the analysis period
Adjusted means: Model-predicted fatalities with is_release_day set to 0 (surrounding) or 1 (release)
Confidence intervals: 95% CIs from OLS standard errors
Placebo test: 1,000 iterations drawing 10 random dates (or random Fridays); count how many produce an effect ≥ observed
Software: Python, statsmodels (OLS via smf.ols), scipy, pandas

This is a straightforward back-of-the-envelope check — we used the same regression specification described in the paper and applied it to the publicly available FARS data. No exotic modeling choices; the goal was to see whether the headline result holds up with one more year of data.

Key Discrepancies

1. Our 2017–2022 reproduction is close but not identical to the paper

  • We find +15.3 excess deaths vs. the paper's +18.2 — roughly 16% smaller
  • Both are statistically significant (p = 0.009 vs. p = 0.01)
  • Likely explained by FARS data revisions between the paper's data pull and ours

2. Adding 2023 collapses the effect

  • Absolute increase drops from +15.3 to +4.1 deaths (−73%)
  • No longer statistically significant (p = 0.47)
  • Placebo performance degrades from 14/1,000 to 162/1,000

3. 2023 albums show no effect at all

  • The five 2023 mega-releases show a negative point estimate (−1.8 deaths)
  • 627 of 1,000 random-date placebos produce a larger "effect"
  • Adjacent-Friday odds ratio is 0.99 — completely flat
